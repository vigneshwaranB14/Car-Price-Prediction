# Car-Price-Prediction
Overview

This project is a Streamlit-based web application that predicts the resale price of a car using machine learning. Users can enter vehicle details such as brand, model, year, fuel type, transmission, and kilometers driven to receive an estimated market price.

Features

Real-time car price prediction
Interactive Streamlit user interface
Data preprocessing and feature engineering
Machine learning model training and evaluation
User-friendly and responsive design

Technologies Used

Python
Streamlit
Pandas
NumPy
Scikit-learn
Machine Learning

How It Works

User enters car details.
Input data is preprocessed.
The trained machine learning model analyzes the data.
The predicted car price is displayed instantly.

Future Improvements

Support for more car brands and models
Advanced price visualization
Integration with live market data
Model performance comparison
